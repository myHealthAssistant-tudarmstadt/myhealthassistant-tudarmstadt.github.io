Enable Third Party Apps Installation

First make sure your Android devices allow the installation of apps from what being refered to as "unknown sources" by default settings.
  • Open phone's Settings menu (usually found by hitting the menu button from the home screen)
  • Tap the option for Applications (or Security for later android versions).
  • Look aroung, you should see an option that says "Unknown sources". Fill the check box next to it, then click OK on the resulting popup alert.
You should now be able to download and install this APK file on your phone. Android should handle the installation automatically after downloaded.
