Enable Third Party Apps Installation

First, make sure your Android device allows the installation of apps from "unknown sources" in the security settings.
  • Open the phone Settings (usually found by hitting the menu button from the home screen).
  • Tap the option for Applications (or Security for later android versions).
  • Scroll to the option "unknown sources" and activiate it, then click OK on the resulting popup alert.
You should now be able to install the APK file on your phone. Android should handle the installation automatically after download.
